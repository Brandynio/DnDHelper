    • Welcome to the DnD Helper!

    This website is a helper tool for playing Dungeons and Dragons.

    Right now we provide you with the ability to create characters, roll dice, search DnD things, and track initiative.

    This tool allows you to create a character. Right now we don't have full character sheets available but the main statistics along with some basic information about the character can be stored. Characters you create will be stored in local storage, meaning your browser will store them on your machine.

    Dice Roller

    With the dice roller, you can roll any amount of dice with any amount of sides. You can also choose whether you want to see the individual rolls, the total, or both!

    Search

    Using the D&D 5e Api we allow you to search for classes, features, and spells!

    Initiative Tracker

    The initiative tracker will help Dungeon Masters keep track of whose turn it is, and how much health everyone in the fight has.

    If you log in and have created characters, you can import those characters into your initiative order.

    Future Plans

    The tools and features above are all we have planned at this moment. However, if we were to continue with this project, here is what we would expand on:

    • Fuller character sheets
    • More search options
    • User accounts
    • Deleting characters
